    About Me

    As a former Chemical Engineer Student, I was introduced to coding by Matlab, a coding class part of my curriculum. I ended up enjoying it immensely so I immediately dove into learning as much as I could. As a result of that effort, I am now a software engineer experienced in JavaScript, React, Redux, Ruby, Rails, HTML, CSS, MongoDB, Node, Express, and R. I've used those skills to create several projects (See below!), including a clone of YouTube and a trivia based game. I am always looking to learn more and strive to continue using my skills to build wonderful experiences for everyone.
